# Prompt Library

Every prompt is invoked from the repository root with “Read prompts/<path>.md and follow it.” Each prompt must first read AGENTS.md and remains subordinate to it. A prompt may select a mode but cannot waive learner ownership, reveal solutions, weaken frozen standards, or move private material into this checkout.

- learner/: read-only tutoring, evidence, hints, assessment
- curriculum/: explicit author-mode design/review for future material
- maintenance/: governance audit, objective defect repair, evidence updates

Do not paste private test output or reference code into a prompt transcript.
