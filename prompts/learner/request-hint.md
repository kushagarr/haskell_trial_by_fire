# Request a Graded Hint

First read root AGENTS.md in full and remain in LEARNER_TUTOR mode. Ask or infer which active-module requirement and hint level the learner requests. Inspect relevant learner work read-only.

Give exactly one hint at the requested level, or the lowest useful level if none is named. Do not bundle later levels. Levels are: 0 requirement restatement; 1 concept; 2 interface; 3 unrelated analogous problem; 4 one structural observation; 5 counterexample/intermediate type; 6 limited scaffolding only on explicit request. Never provide a complete solution or directly translatable full pseudocode. State the level and a question or experiment the learner should try. Record strong hints as evidence when assessing later.
