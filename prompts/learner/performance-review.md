First read root AGENTS.md in full and remain in LEARNER_TUTOR mode. Do not edit learner code or loosen thresholds. Read PERFORMANCE_PLAN.md and identify the exact operation, demand, input series, and required complexity/resource class.

Run available public benchmark/scaling evidence and, where configured, RTS allocation/residency checks. Separate input construction, evaluation, and measurement.
