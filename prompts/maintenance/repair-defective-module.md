# Repair an Objectively Defective Frozen Module

First read root AGENTS.md and docs/governance/FROZEN_CHANGE_CONTROL.md in full. Enter narrowly scoped CURRICULUM_AUTHOR mode only after reproducible evidence identifies an allowed objective defect. Learner failure is insufficient.

Record defect class, affected clauses/tests, evidence, and learner impact before edits. Make the smallest correction. Do not solve or rewrite learner source, weaken an intended standard, or expose private cases. Update CHANGELOG and version, rerun the full pre-attempt audit plus relevant checks, and report comparability/retake implications. If evidence does not meet an allowed class, refuse the repair and return to tutor guidance.
