# Transparent Support Commands

support/bin/apprentice is a small dispatcher, not a teaching framework. It prints or runs ordinary Cabal/audit commands so the learner can see the underlying operation.

~~~text
./support/bin/apprentice build
./support/bin/apprentice test M001
./support/bin/apprentice assess M001
./support/bin/apprentice bench M001
./support/bin/apprentice review M001
./support/bin/apprentice governance-audit
./support/bin/apprentice integrity
~~~

Assessment remains defined by ASSESSMENT.md and module metadata. Private CI is external.
